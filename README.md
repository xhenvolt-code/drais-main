# Ibun Baz Girls Secondary School - DRAIS

A comprehensive school management system (DRAIS - Digital Resource and Academic Information System) built with Next.js for Ibun Baz Girls Secondary School.

## Features

- Student Management
- Staff Management  
- Attendance Tracking (Biometric & Manual)
- Finance & Fee Management
- Academic Results
- Tahfiz (Quran Memorization) Tracking
- Timetable Management
- Reports & Analytics

## Tech Stack

- **Frontend**: Next.js 14 (App Router), React, TypeScript
- **Backend**: Next.js API Routes
- **Database**: MySQL
- **Authentication**: JWT with fingerprint/biometric support

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Configure database connection in environment variables

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## School Information

- **Name**: Ibun Baz Girls Secondary School
- **Location**: Busei, Iganga along Iganga-Tororo highway
